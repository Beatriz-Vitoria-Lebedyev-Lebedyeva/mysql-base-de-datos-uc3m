# mysql-base-de-datos-uc3m
Proyecto académico desarrollado para la asignatura de Bases de Datos de la Universidad Carlos III de Madrid. El objetivo consistió en diseñar e implementar una base de datos relacional para la gestión de eventos científicos, siguiendo unos requisitos funcionales definidos por la asignatura.

## Tecnologías
- MySQL
- SQL

## Contenido
- `Create_BD.sql` – Creación de las tablas que modelan la información del sistema, incluyendo claves primarias, claves foráneas y restricciones de integridad.
- `Insert_BD.sql` – Inserción de datos sintéticos para realizar pruebas.
- `queries_BD.sql` – Consultas SQL para el análisis y explotación de la información.

## Consultas implementadas
Entre otras:
- Número de eventos por tipo y año.
- Eventos con mayor financiación.
- Investigadores con mayor actividad organizativa.
- Eventos por categorías temáticas.
- Estadísticas por departamentos y centros.
- Participación de investigadores y personal externo.

## Competencias
- Diseño de bases de datos relacionales.
- Modelado de entidades y relaciones.
- Desarrollo de consultas SQL.
- Extracción y análisis de información.

Nota: Proyecto realizado en pareja con fines exclusivamente académicos. Los datos utilizados son sintéticos y fueron generados siguiendo las especificaciones de la práctica.
